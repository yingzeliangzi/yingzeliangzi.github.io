新版Steam更新后 在Steam或SteamLauncher快捷方式后添加-vgui参数可以回到旧版UI 正常使用皮肤

Created2023-06-27|Updated2023-06-27
|Word count:534|Reading time:1min|Post View:

此举例使用 Jack-Myth/Threshold-Miku 皮肤

右键 Steam 或 SteamLauncher 属性

SteamLauncher.exe 是之前 Steam 修改了库的 css 代码 导致无法正常使用皮肤 大佬做的皮肤启动器
直接在目标后面加空格 输入-vgui 如下

如果是原版 Steam 快捷方式 也同理

示例图 包括 Steam 登录/Steam 商店都是原版 UI 但是退出的动画是新版 UI

附加: 如何开机启动带皮肤的 Steam

前因

平时打游戏爱好为主 所以开机选择自动启动 Steam 虽然内存占用多一点 有时候达到 300M 但是不怎么吃 CPU 就一直放着了
但是如果是加了启动项 和 使用大佬自制皮肤启动器 该如何设置呢?

解决方案

使用 Windows 自带的计划任务(Task Scheduler)
右键此电脑-管理(This PC-Manage) 点进 Task Scheduler Library

选择新建任务(Create Task…)

General

名称和描述随便填写 自己认识即可 无需英文
安全选项第一个即可 当你的用户登录时

Triggers

点击下面的新建(New…)
在最上面选择触发条件 在用户登录时(At log on)
下面的启用(Enable)会自动勾选

然后点击确定

Actions

点击下面的新建(New…)
选择默认的 Action-打开一个应用(Start a program)
程序或脚本选择 Steam 或 SteamLauncher(选择快捷方式也可以)
下面的附加选项(Arguments)填入”-vgui”无需空格

最后确定创建 记得在任务管理器-启动或者 Steam 内关闭它的自动启动(不是我们创建的这个任务)
后面的选项不用管了 如果出现笔记本电脑有时候电池无法自动启动 Steam 的话
检查创建任务的第四步(Conditions) 文中没有提到
取消勾选电源里面的如果电脑转为电池供电则停止任务(Stop if the computer switches to battery power)
如果不想开机启动 再来计划任务把你之前设置的名称指定的任务删除即可 或者 disable 掉
